The SD-card image can have 4 files called PART1.po to PART4.po each file will be loaded as a 32meg disk. Or use the better alternative of a config file. Create a file called config.txt in the root of the sd card. The contents of "config.txt" must have a file name written on each line, and you can specify up to four files to launch. One file per line in the text file. The filenames can include a path, keep the whole name to less than 99 characters long.

You can add as many files as you like up to the capacity of the SD card. However, only the four files with the names written in config.txt will be used. The file extensions are either ".po", ".hdv", or ".2mg". Please keep in mind that the file specification must always be 4 lines for 4 files, no blank lines and no comments etc.

You can keep several text files on the sd card and rename them to switch between different groups of files. Only the one called config.txt is live.
